# 分娩期アセスメント早見表

助産学の学習・臨床実習向けに作成した、分娩期の異常アセスメントをすぐに確認できるWebアプリです。
項目をタップするだけで「原因 → 観察項目 → 母児への影響 → できるケア」を一気に確認できます。

## 概要

分娩中に起こりうる異常を一覧化し、それぞれについて臨床で押さえておきたい4つの視点を素早く参照できるようにしたリファレンスツールです。スマートフォンでの使用を想定し、iPhoneのホーム画面に追加すればアプリのように使えます。

## 収録項目（12種類）

| No. | 項目 |
|---|---|
| 1 | 微弱陣痛 |
| 2 | 過強陣痛 |
| 3 | 遷延分娩 |
| 4 | 胎児機能不全 |
| 5 | 常位胎盤早期剥離 |
| 6 | 臍帯下垂・脱出 |
| 7 | 弛緩出血 |
| 8 | 回旋異常 |
| 9 | 前期破水（PROM） |
| 10 | 肩甲難産 |
| 11 | 羊水塞栓症 |
| 12 | 子宮破裂 |

## 主な機能

- **早見表示**：項目をタップすると詳細カードが表示され、自動でその位置までスクロール
- **アコーディオン展開**：「原因」「観察項目」「母児への影響」「できるケア」はタップした項目だけが開く仕様
- **内容の編集・保存**：鉛筆アイコンから各項目の内容を自由に書き換え、端末内に保存可能
- **初期状態へのリセット**：編集した内容をワンタップで元に戻せる
- **ホーム画面追加対応**：iPhoneのSafariから「ホーム画面に追加」でアプリのように利用可能

## 使い方

1. 上部の分類ボタン（1〜12）から確認したい項目をタップ
2. 表示されたカードの「原因」「観察項目」などの見出しをタップすると内容が開く
3. 鉛筆アイコンをタップすると編集モードになり、内容を書き換えて「登録する」で保存
4. フッターの「編集内容をすべて初期状態に戻す」から、いつでもリセット可能

## 技術情報

- 単一のHTMLファイル（`index.html`）で構成。外部サーバー不要で、GitHub Pagesでそのまま公開可能
- 編集内容はブラウザの `localStorage` に保存されるため、**保存データは使用した端末・ブラウザのみに反映**されます（他の端末とは共有されません）

## 免責事項

本アプリは学習用の早見表であり、臨床判断の代替にはなりません。実際の臨床では必ず医師・指導者の判断を優先し、各施設のプロトコルや最新のガイドラインを確認してください。
